## 核心论点

5 个我每天都在用的自定义命令。代码 + 用法 + 为啥这 5 个活下来了。

## 钩子候选

- Claude Code 的 /custom command 是隐藏的生产力加速器。我的 Top 5。
- 写过 15 个自定义命令，只有 5 个活到今天。为啥？

## 主干

- **/commit-with-my-style**：按我个人 commit message 风格写。省 30 秒/commit
- **/review-changes**：过一遍 git diff，给 code smell 提醒
- **/explain-this-error**：粘错误日志，它按我项目上下文定位根因
- **/migrate-file**：把一个文件从 old pattern 重构到 new pattern（带约束）
- **/find-bug**：给一个症状描述，它从 recent commits 里猜

每个命令：1-2 行配置代码 + 一个真实用例

## 结论候选

- 真正的工具效率不在主菜单，在你自定义的那几项。
