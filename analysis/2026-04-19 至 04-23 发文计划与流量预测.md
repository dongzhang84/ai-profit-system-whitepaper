## 三、逐篇流量预测

### 1. 黄仁勋 / 芯片封锁（周日发）

- **题材优势**：AI + 中美政治双热点，话题本身自带流量。黄仁勋是华语圈认脸度最高的 AI 名人之一。
- **身份锁定**：中国科技从业者、关心国运的普通读者。基数大。
- **风险**：话题政治敏感，部分读者会出于立场划走。"主持人吵红了脸"这种标题可能被部分人判定为标题党。
- **预测区间**：**2.5k - 5k**。目标 3k。如果黄仁勋本人或大 V 转，上 5k 也不意外。

### 2. Cursor 套壳公司（周一发）

- **题材优势**：Cursor 最近估值暴涨，行业内讨论密集。"290 亿美元 vs 被房东杀死"反差强。
- **身份锁定**：创业者、科技投资人、AI 圈从业者。基数中等。
- **风险**：AI 圈内话题，出圈能力不如"普通人也关心"的题材。"套壳"一词可能引起部分 Cursor 粉丝反感。
- **预测区间**：**1.5k - 3k**。目标 2k。

### 3. SpaceX 立志传（一）（周二发）

- **题材优势**：SpaceX + 马斯克 + 太空，华语圈基本盘稳定。上一篇《重返星辰大海》4.6k，证明太空话题有需求。
- **风险**：系列第一篇，读者可能不愿 commit 往后看；内容是历史回顾而非即时热点，情绪钩子弱于时事。
- **预测区间**：**1.5k - 3k**。目标 2k。这篇主要是**测试系列开篇的单篇独立性**。如果第一篇能打到 2k，后续第二到第五篇可以继续发。
- **次级目标**：后台观察"系列追读率"，如果这篇流量尚可但完读率低，说明系列 framing 需要调整。

### 4. 卡拉比—丘成桐（下）（周三发）

- **题材短板**：纯数学物理科普，系列下篇。X 华语圈受众极窄。
- **发布动机**：**知乎收尾，不指望 X 流量**。上中篇已发知乎，这篇的主要使命是结束烂尾系列，给知乎读者一个交代。
- **预测区间**：**X 流量 0.5k - 1.2k**（大概率是本周最低）。**目标：知乎完整度，不看 X 数字**。
- **操作建议**：X 发布时文案里明说"这是为知乎系列收尾"，管理读者预期，降低对 X 流量的焦虑。

### 5. Vibe Writing 宣言（周四发）

- **题材优势**：创作者元话题。类似"为 X 打工"那篇（3.8k），目标读者本身就是 X 重度用户，转发动力强。
- **身份锁定**：所有想在 X 写长文的人，基数中大。
- **风险**：标题带"宣言"容易显得中二，点击率看执行。周四发，国内周五早上读，接近周末流量略弱。
- **预测区间**：**2k - 4k**。目标 2.5k。**有机会是本周次高**。

## 五、风险点

1. **周日热点的时效性**。黄仁勋如果在周五周六被国内大 V 讲过一轮，周日再发可能已经过了热点峰值。周六下午应快速扫一眼国内微博热搜和华语 X 圈。如果已被讲烂，考虑换到周一发或调整标题切入角度。

2. **卡拉比在 X 上拉低均值**。如果不想让这篇稀释整体数据感，可以在发布文案里明说"这是给知乎读者的收尾篇，不追求 X 流量"，提前管理读者预期。

3. **SpaceX 系列首篇不成**。如果周二这篇低于 1.5k，后续第二到第五篇需要重新 framing 再发，不要一篇一篇地往下掉。
